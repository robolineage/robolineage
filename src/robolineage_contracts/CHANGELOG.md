# robolineage_contracts Changelog

This public release keeps the active contract package focused on the current RoboLineage lifecycle:

- core rollout and metadata contracts;
- session state and event contracts;
- VSA snapshot and validation contracts;
- dataset lock, train manifest, and policy metadata contracts.

Older planning notes, branch-specific implementation logs, and legacy design references were removed from the public package. Contract changes should now be documented through concise release notes and schema version bumps.
